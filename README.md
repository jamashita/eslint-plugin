ESLint Library bundle for us.

## Rules

### `@jamashita/eslint-plugin`

#### Install

```text
yarn add @jamashita/eslint-plugin \
  eslint \
  typescript -D
```

### Append in `.eslintrc`

```json
{
  "extends": [
    "plugin:@jamashita/eslint-plugin/recommended"
  ]
}
```

or

```json
{
  "extends": [
    "plugin:@jamashita/recommended"
  ]
}
```
